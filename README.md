# Simple React App

Simple React Web App with 3 page using `react-router-dom` npm package and the `bem convention` for CSS.

### Install with SSH:

> $ git clone git@github.com:gael-src/simple-react-app.git

> $ cd simple-react-app

> $ npm install

> $ npm start
